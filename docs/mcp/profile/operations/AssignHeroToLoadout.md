# AssignHeroToLoadout
Assign a hero to a loadout by ID.

## Attributes
Compatible Profiles: `campaign`  
Supported MCPs: `fortnite`

## Payload
```json
{
    "slotName": "",
    "loadoutId": "",
    "heroId": ""
}
```

## Parameters
- `slotName`: name of slot
- `loadoutId`: ID of loadout to assign hero to
- `heroId`: ID of hero
