# EquipCampaignCustomization
Equips a customization in Fortnite: Save the World.

## Attributes
Compatible Profiles: `campaign`  
Supported MCPs: `fortnite`

## Payload
```json
{
    "slotName": "",
    "itemToSlot": ""
}
```

## Parameters
- `slotName`: enum `PersonalVehicle`
- `itemToSlot`: ID of item in slot
