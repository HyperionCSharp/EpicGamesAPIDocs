# SetMtxPlatform
Set the current MTX platform used for purchases in the item shop.

## Attributes
Compatible Profiles: `common_core`  
Supported MCPs: `fortnite`

## Payload
```json
{
    "newPlatform": ""
}
```

## Parameters
- `newPlatform`: the platform that will be used for purchases
   - Epic
   - PSN
   - Live
   - Shared
   - EpicPC
   - EpicPCKorea
   - IOSAppStore
   - EpicAndroid
   - Nintendo
   - WeGame
   - Samsung
