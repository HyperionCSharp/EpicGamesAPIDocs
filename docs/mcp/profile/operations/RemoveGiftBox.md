# RemoveGiftBox
Removes a list of gift boxes from a player's inventory.

## Attributes
Compatible Profiles: `common_core`  
Supported MCPs: `athena`

## Payload
```json
{
    "giftBoxItemIds": []
}
```

## Parameters
- `giftBoxItemIds`: list of gift box GUIDs in inventory (e.g. `ea74a9cc-a0d5-4625-adcb-9e8c5ff9c256`)
