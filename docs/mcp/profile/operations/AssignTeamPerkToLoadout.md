# AssignTeamPerkToLoadout
Assign a team perk to a loadout by ID.

## Attributes
Compatible Profiles: `campaign`  
Supported MCPs: `fortnite`

## Payload
```json
{
    "teamPerkId": "",
    "loadoutId": ""
}
```

## Parameters
- `teamPerkId`: ID of team perk
- `loadoutId`: ID of loadout to assign team perk to
